ConsolidDesign Assistant

A CAD Tool to Support Part Consolidation Design for Additive Manufacturing



Project Description :

Additive manufacturing (AM) has extremely expanded and refreshed the way how product is designed and made. Part consolidation is one of these new opportunities. Against the conventional philosophy of segmenting a complex design into multiple simple parts and then rejoined with welding or bolts, part consolidation facilitates "complexity for free" mindset and helps designers be focused on "design for function" rather than "design for manufacture and assembly". The main challenges include:

  • Part Consolidation Candidacy Detection (PCCD)
  • Fusion of parts with AM characteristics

Introduction of ConsolidDesign

The ConsolidDesign Assistant is a developed plugin for Rhinoceros in support of part consolidation redesign. The main program is written in C# language. User interface (UI) and data access are derived from the built-in classes and methods from Microsoft .NET framework. Extracted data from the design tool is saved and read by following the Extensible Markup Language (XML) schema. For geometric and material information of the CAD model, it is accessed and manipulated by RhinoCommon.

The tool has four modules:
1. Info setup: user
2. Screening: user
3. Embodiment: user
4. Optimization: user

The homogenization model can be used in the initial design stage to help designers to get the effective mechanical property of any lattice topology. Designers don't need a solid mathematical background. The effective property can by simply obtained by the homogenization code in SimuLattice.

The joint stiffening element model can be used in the design optimization stage. It can consider the influence of the joint on the stiffness of the lattice structure. Therefore, the result from the joint stiffening element model is more accurate.

The hybrid element model can be used to simulate the solid-lattice hybrid structure. It uses the beam element to model the lattice structure; it uses the tetrahedral element to model the solid part. The total computational cost of the hybrid element model is much lower than the traditional model.

The as-built material property model is to consider the manufacturing influence of AM on the mechanical property of lattice structures. The material data is obtained experimentally. Interpolation functions are used to construct the as-built material property model. Currently the material data stored in SimuLattice is for PLA fabricated by Ultimaker 3. More materials will be added in the future.
