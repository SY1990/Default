SimuLattice

A simulation software to support the design of lattice structures fabricated by Additive Manufacturing



It has four models:

  • 3D numerical homogenization model
  • Joint stiffening element model
  • Hybrid element model
  • As-built material property model

Introduction

The design of lattice structure can be divided into three design stages:

1. Initial design stage
2. Design optimization stage
3. Design validation stage

The homogenization model can be used in the initial design stage to help designers to get the effective mechanical property of any lattice topology. Designers don't need a solid mathematical background. The effective property can by simply obtained by the homogenization code in SimuLattice.

The joint stiffening element model can be used in the design optimization stage. It can consider the influence of the joint on the stiffness of the lattice structure. Therefore, the result from the joint stiffening element model is more accurate.

The hybrid element model can be used to simulate the solid-lattice hybrid structure. It uses the beam element to model the lattice structure; it uses the tetrahedral element to model the solid part. The total computational cost of the hybrid element model is much lower than the traditional model.

The as-built material property model is to consider the manufacturing influence of AM on the mechanical property of lattice structures. The material data is obtained experimentally. Interpolation functions are used to construct the as-built material property model. Currently the material data stored in SimuLattice is for PLA fabricated by Ultimaker 3. More materials will be added in the future.
